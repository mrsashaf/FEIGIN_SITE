[ 05 / LOCKED RULES ]

Правила

Эти правила не подлежат изменению. Любая новая страница FEIGIN должна им следовать.

[ DO ] ШРИФТЫ
Только Inter, Roboto Mono и Panchang 900. Любой другой шрифт — нарушение системы.
[ DO ] ЦВЕТА
Только палитра из раздела 04. Серая иерархия для понижения важности. Красный #ff4444 — единственный акцент.
[ DO ] CLAMP() ДЛЯ ЗАГОЛОВКОВ
Все hero и section-titles — через clamp(min, vw, max). Это убирает нужду в десятке media-queries.
[ DO ] ПЕРЕИСПОЛЬЗОВАНИЕ
Новый контент = новые слова в существующих компонентах. Не изобретать новые рамки, штампы, плашки.
[ DON'T ] НЕ СМЕШИВАТЬ ШРИФТЫ
Mono — только tech. Inter — только body и заголовки. Panchang — только логотип ABERRATION. Никогда не наоборот.
[ DON'T ] НЕ ВВОДИТЬ ЦВЕТА
Никаких синих, зелёных, жёлтых, фиолетовых. Никаких новых градаций серого помимо перечисленных.
[ DON'T ] НЕ POSITIVE TRACKING НА BODY
Положительный letter-spacing — только для UPPERCASE мелких текстов (≤16px). На обычном body — никогда.
[ DON'T ] НЕ ТОНКИЕ ЗАГОЛОВКИ
Section titles — минимум вес 700. Тонкие веса (300–400) разрешены только для footer-links и anthology-text.
[ 06 / BREAKPOINTS ]

Адаптив

Mobile-first. Размеры через clamp() — media-queries только для смены layout.

DESKTOP (DEFAULT)
≥ 1025px
Полный layout: 2-колоночные cases, 3-колоночный evidence-grid с masonry-смещением.
@media (max-width: 1024px)
≤ 1024px
Tablet: уменьшен padding header и секций. evidence-grid ↓ 2 колонки.
@media (max-width: 991px)
≤ 991px
evidence-grid: 2 колонки с masonry-смещением 80px.
@media (max-width: 768px)
≤ 768px
Mobile: header ↓ column, все grid ↓ 1 колонка, форма clearance ↓ column, печать ABERRATION на весь экран.
@media (max-width: 420px)
≤ 420px
Small phones: уменьшены main-title, lore-text, access-card-title для безопасного fit.
